This plugin will provide the Dutch translation for your bbPress installation.

== Description ==

We've got tired with bbPress losing its manually uploaded translations files whenever there's and update. This is a temporary plugin until language packs for plugins has been sorted.

== Installation ==

Install like you would with any other WordPress plugin

1. Upload the bbPress-NL.zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Who is providing the translations? =

Glad you asked. It's provided graciously by anyone and everyone who contributes on the [bbPress translation project on GlotPress](http://translate.wordpress.org/projects/bbpress).
